## 1. 一个故事：为什么需要审评员指南？

### 审评员的困境

想象你是 FDA 的审评员，收到了一家制药公司的新药申请：

```
收到的数据包
────────────
├── sdtm/
│   ├── dm.xpt（患者基本信息）
│   ├── vs.xpt（生命体征）
│   ├── ae.xpt（不良事件）
│   ├── lb.xpt（实验室检查）
│   ├── ... 20+ 个数据文件
│   └── define.xml
│
└── adam/
    ├── adsl.xpt（受试者级别数据）
    ├── advs.xpt（生命体征分析数据）
    ├── adae.xpt（不良事件分析数据）
    ├── ... 10+ 个数据文件
    └── define.xml
```

**你的问题：**
- 应该从哪个文件开始看？
- 这个研究有什么特殊的地方需要注意？
- 数据有没有已知的问题？
- 为什么某些变量的值和标准不同？
- 如何验证分析结果的正确性？

**没有"导游"，你会在数据海洋中迷失！**

### 审评员指南就是"数据导游"

```
审评员指南的作用
────────────────

就像：
• 旅游时的导游 → 告诉你去哪里看什么
• 博物馆的导览册 → 告诉你展品的背景
• 软件的用户手册 → 告诉你怎么使用
• 考试的答题说明 → 告诉你怎么理解题目

审评员指南：
• 告诉审评员数据的概况
• 告诉审评员从哪里找什么信息
• 告诉审评员有什么需要注意的地方
• 帮助审评员更高效地审查数据
```

## 2. SDRG 和 ADRG 一句话解释

### SDRG

> **SDRG = Study Data Reviewer's Guide (for SDTM) = SDTM 审评员指南，帮助审评员理解和使用 SDTM 数据的导航文档。**

### ADRG

> **ADRG = Analysis Data Reviewer's Guide = ADaM 审评员指南，帮助审评员理解和使用 ADaM 数据的导航文档。**

#### 第 5 节：派生变量和分析规则

```
派生变量和分析规则示例
────────────────────────

5.1 基线定义

对于 ADVS（生命体征）：
    基线 = 首次给药日期（TRTSDT）前最后一次非空测量

    算法：
    1. 筛选 ADT < TRTSDT 的记录
    2. 按 ADT 降序排列
    3. 取第一条非空 AVAL 的记录
    4. 标记该记录 ABLFL = Y
    5. BASE = 该记录的 AVAL

对于 ADLB（实验室检查）：
    基线 = 筛选期或治疗前的最后一次非空测量
    （与 ADVS 相同的逻辑）

5.2 变化值计算

CHG = AVAL - BASE

条件：
• 仅在 ABLFL ≠ Y 的记录上计算
• 如果 AVAL 或 BASE 缺失，CHG 为空

PCHG = (AVAL - BASE) / BASE × 100

条件：
• BASE ≠ 0
• 如果 BASE = 0 或缺失，PCHG 为空

5.3 分析访视窗口

目标访视         时间窗口定义              AVISIT
──────────────────────────────────────────────────────────────
基线             TRTSDT 前 14 天内         Baseline
第 1 周          TRTSDT 后 4-10 天         Week 1
第 4 周          TRTSDT 后 25-35 天        Week 4
第 8 周          TRTSDT 后 53-63 天        Week 8
第 12 周         TRTSDT 后 81-91 天        Week 12

窗口规则：
• 如果多个测量落入同一窗口，取最接近目标日的测量
• 窗口外的测量不纳入主要分析

5.4 缺失数据处理

主要分析：末次观察值结转法（LOCF）
• 如果某访视缺失，用前一访视的值填补
• 标记 LOCF = Y

敏感性分析：完整病例分析（不填补）
```
